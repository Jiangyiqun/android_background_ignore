## 网络不稳定
使用数据网络时，关闭Wifi，以避免自动连上无效的公共WIfi（如酒店、麦当劳、机场提供的 Wifi）。推荐[修改网络验证服务器](https://github.com/Jiangyiqun/android_background_ignore/tree/master/captive_portal_server_changer)自动检查 WIfi 有效性。
